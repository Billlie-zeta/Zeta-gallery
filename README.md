# 사용 가이드

다크 글래스 캐릭터 일러스트 갤러리입니다.
**이미지를 폴더에 넣기만 하면** 자동으로 올라옵니다. 검색·태그 필터·최신순 정렬 포함.

---

## 처음 한 번만: 깃허브에 올리고 켜기

1. 깃허브에서 새 저장소를 **Public**으로 만듭니다.
2. 이 폴더의 파일 전부를 올립니다. (`.github` 폴더, `.gitignore` 포함)
3. **Settings → Pages → Source** 를 **GitHub Actions** 로 선택합니다.
4. 1~2분 뒤 `https://<아이디>.github.io/<저장소이름>/` 에서 열립니다.

> 이미지를 올릴 때마다 자동으로 다시 빌드되어 반영됩니다.

---

## 매일 쓰는 법

### 새 일러스트 추가
`images` 폴더 안에 **캐릭터 이름으로 폴더**를 만들고 이미지를 넣습니다.

```
images/
 └─ 차서진(금연 중인 연인이 전여친에게 간 이유)/
     ├─ 1.png
     └─ 2.png
```

- **폴더 이름이 메뉴가 됩니다.** 괄호를 쓰면 앞은 크게(이름), 괄호 안은 작게(부제)로 자동 분리돼요.
  예: `차서진(금연 중인 연인이 전여친에게 간 이유)` → **차서진** / 아래에 작게 *금연 중인 연인이…*
- 괄호 없이 `차서진`만 써도 됩니다. (부제 없이 이름만)
- 지원 형식: png, jpg, jpeg, webp, gif, avif
- 정렬은 **파일명 순서**를 따릅니다. 최신 컷이 위로 오게 하려면 파일명 앞에 날짜나 번호를 붙이세요 (예: `2026-07-11_re.png`, `03_밤.png`). 갤러리 상단의 정렬 버튼으로 오름/내림을 바꿀 수 있어요.

### 제목·메모·태그 달기 — 전부 선택
`info.json` 을 열어서, 쓰고 싶은 이미지에만 적으세요. **안 적으면 그림만 깔끔하게 뜹니다.**

```json
{
  "차서진(금연 중인 연인이 전여친에게 간 이유)": {
    "1.png": {
      "title": "재회의 밤",
      "memo": "본편엔 못 넣은 컷. 이 대사 장면이 제일 애틋하다.",
      "tags": ["#감정신", "#재회"]
    },
    "2.png": { "title": "손끝의 온도" }
  }
}
```

- **폴더 이름 키**는 실제 폴더명과 똑같이. **파일명 키**도 실제 파일명과 똑같이.
- `title` : 카드와 팝업 위에 뜨는 제목
- `memo` : 팝업에서 이미지 아래 문단으로
- `tags` : 검색·태그 필터에 사용 (예: `["#러프","#컬러"]`)
- 셋 다 선택이라, 원하는 것만 골라 쓰면 됩니다.

### 제목·부제 바꾸기
`index.html` 맨 위 `SITE` 부분의 `brand`, `title`, `subtitle` 세 줄만 고치면 됩니다.

### 색·폰트 바꾸기
`index.html` 의 `:root` 부분 색상값을 바꾸면 됩니다.

## 방문자 분석 붙이기 (선택)

홍보했을 때 사람이 들어오는지 보고 싶으면:

1. goatcounter.com 가입 → 사이트 코드 하나 정함
2. 받은 `<script>` 한 줄을 복사
3. `index.html` 에서 `<!-- GOATCOUNTER: 여기에 붙여넣기 -->` 줄 바로 아래에 붙여넣고 커밋
4. `<내이름>.goatcounter.com` 대시보드에서 방문자·유입 경로 확인

안 붙여도 갤러리는 정상 작동합니다.

## 알아둘 점

- **저장소가 Public이라 이미지·소스는 링크를 아는 누구나 볼 수 있습니다.** (무료 플랜 특성)
- 노골적 성인 콘텐츠는 깃허브 정책상 올리면 안 됩니다. 복근·노출 의상 정도는 문제없습니다.
- `gallery.json` 은 배포할 때 자동 생성되니 직접 건드릴 필요 없습니다.
- 무료 한도: 사이트 1GB, 월 트래픽 100GB. 웹용 이미지 수백 장은 여유입니다.
